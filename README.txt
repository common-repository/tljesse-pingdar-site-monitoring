Monitor your WordPress site availability with Pingdar monitoring services.

== Description ==

Know when your site goes down!  [Pingdar](https://pingdar.com/) monitors your site and notifies you if your site is not reachable for any reason.  In the free version, your site is checked every 10 minutes.   If your site is not pingable then you will recieve both an email and an sms message.

== Installation ==

1. Upload the `pingdar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register or sign in to Pingdar from the 'Pingdar Admin' in WordPress admin

== Frequently Asked Questions ==

= Does Pingdar cost money? =

This version of the Pingdar plugin works with the free service. A later release of the plugin will include the premium Pingdar monitoring services as well.

= What does the free service monitor? =

You'll receive a constant log of ping response times displayed so you can tell when your site has had an outage.
